O sexto dia é dedicado à **Base de Dados**. Aprendemos a modelar dados, criar esquemas relacionais e realizar consultas. Uma base de dados bem estruturada é o alicerce de qualquer aplicação robusta, permitindo o armazenamento persistente e a recuperação rápida de informação.

## Foco
- Base de dados e integração backend

## Self-learning
- SQLite
- Tabela 
- Chave primária
- Ligação entre backend e base de dados

## Tarefas de desenvolvimento
- Criar schema da Bd
- Implementar operações básicas de criação e linguagem
- Persistir dados reais

## Entregável
- Backend persistente com SQLite
