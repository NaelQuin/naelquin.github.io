# User Stories

As user stories estão organizadas por perfil de utilizador, com prioridade segundo a escala MoSCoW (**Must / Should / Could**).

| ID    | Role                    | User Story                                                                                                                                                                           | Priority | Acceptance Criteria                                                                                                                                                                               |
|-------|-------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| US-01 | Investigador / Analista | Como investigador, quero visualizar a distribuição sociodemográfica da amostra (género, faixa etária, habilitações, situação profissional, bairro) para caracterizar os 59 participantes. | Must     | Dashboard exibe gráficos de distribuição para cada variável sociodemográfica; valores coincidem com o relatório (ex: 55,9% feminino; pico etário nos 41-50 anos).                                 |
| US-02 | Investigador / Analista | Como investigador, quero comparar as médias das escalas SC e EC por situação profissional, para identificar os grupos com maior e menor predisposição cívica.                         | Must     | Gráfico agrupado exibe médias de SC e EC por categoria profissional; é possível ordenar e filtrar; valores globais visíveis (SC=3,77; EC=4,63).                                                   |
| US-03 | Investigador / Analista | Como investigador, quero visualizar os níveis de conhecimento e participação nos 19 projetos do programa "Comunidades em Ação", ordenados por notoriedade.                            | Must     | Gráfico stacked bar exibe as três categorias (não ouvi / ouvi / participei) para cada projeto; projetos ordenáveis por % de conhecimento; Reabilitação EB Nº1 aparece no topo (76,3%).           |
| US-04 | Investigador / Analista | Como investigador, quero filtrar os dados por bairro de residência para explorar diferenças territoriais nos padrões de participação.                                                 | Should   | Filtro interativo por bairro disponível na dashboard; todos os gráficos atualizam ao selecionar um bairro; opção "Todos" disponível por defeito.                                                  |
| US-05 | Investigador / Analista | Como investigador, quero ver a distribuição dos três perfis de participação (Baixo, Intermédio, Avançado) cruzados com variáveis sociodemográficas e escalas.                         | Should   | Gráfico de distribuição dos perfis exibido (Baixo=33,9%; Avançado=5,1%); possibilidade de cruzar perfil com género, faixa etária e bairro; médias SC e EC visíveis por perfil.                   |
| US-06 | Gestor OIL / Técnico    | Como gestor da OIL, quero um painel de visão geral com os principais KPIs do território (N participantes, % interesse em participar, médias globais SC e EC, % perfil avançado).     | Must     | Painel de topo com 5 cartões KPI: total de participantes (59); % interesse "Sim" (76,3%); SC Global (3,77); EC Global (4,63); % Perfil Avançado (5,1%); valores atualizados por filtro de bairro. |
| US-07 | Gestor OIL / Técnico    | Como gestor, quero ver quais as áreas temáticas de maior interesse dos moradores para orientar as próximas iniciativas comunitárias.                                                  | Must     | Gráfico de barras horizontal com áreas ordenadas por frequência; Limpeza/espaços verdes (23) e Habitação (21) no topo; possibilidade de filtrar por situação profissional.                       |
| US-08 | Gestor OIL / Técnico    | Como gestor, quero identificar os canais de comunicação mais eficazes para planear campanhas de mobilização comunitária.                                                              | Must     | Gráfico de barras com ranking de canais (Cartazes 54,2%; WhatsApp 40,7%; Telefone 39,0%; Redes Sociais 32,2%); exportável como imagem.                                                            |
| US-09 | Gestor OIL / Técnico    | Como gestor, quero ver os principais motivos de não participação para desenhar estratégias de remoção de barreiras.                                                                   | Should   | Gráfico de barras com motivos ordenados por frequência; "Não sei como me envolver" (3) destacado no topo; acompanhado de nota contextual.                                                         |
| US-10 | Gestor OIL / Técnico    | Como gestor, quero ver as condições que os moradores mais valorizam para se envolverem, para priorizar o que oferecer primeiro.                                                       | Should   | Gráfico de barras horizontal com condições ordenadas; "Acesso a informação" (33) e "Acesso a apoios/recursos" (28) no topo; filtrável por bairro.                                                 |
| US-11 | Decisor / Stakeholder   | Como decisor, quero exportar um relatório em PDF com as métricas chave e gráficos da dashboard para partilhar com parceiros e financiadores.                                         | Could    | Botão de exportação PDF disponível na dashboard; relatório exportado inclui KPIs, gráficos principais e título do projeto; formato legível e sem elementos cortados.                              |
| US-12 | Decisor / Stakeholder   | Como decisor, quero comparar as dimensões atitudinais vs. comportamentais do EC para compreender o gap entre intenção e ação dos moradores.                                           | Must     | Gráfico side-by-side exibe EC Atitudes (5,27) vs. EC Comportamentos (3,77); diferença destacada visualmente; acompanhado de nota interpretativa sobre o gap.                                     |
